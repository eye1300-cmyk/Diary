🚀 적용 방법 (한 번만 하면 됩니다)

1
원하는 아이콘 다운로드 → 파일명을 apple-touch-icon.png 으로 바꾸기
2
GitHub 저장소에 apple-touch-icon.png 업로드 (diary.html과 같은 폴더)
3
아이폰 사파리에서 일기장 주소 열기 → 하단 공유 버튼(□↑) → 홈 화면에 추가
4
홈 화면에 예쁜 아이콘이 생겼습니다 ✓
💡 이미 홈 화면에 추가했다면? 기존 아이콘을 길게 눌러 삭제한 뒤 다시 추가해야 새 아이콘이 반영됩니다.
