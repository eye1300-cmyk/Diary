☁️ Firebase 동기화 설정

여러 기기에서 일기를 공유하려면 Firebase 설정이 필요합니다.
한 번만 설정하면 됩니다.

  1. console.firebase.google.com → 새 프로젝트 만들기
  2. Realtime Database 생성 → 규칙 탭 → 아래 내용으로 교체 후 게시:
    { "rules": { ".read": true, ".write": true } }
  3. ⚙️ 프로젝트 설정 → 내 앱 → 웹() 앱 추가 → firebaseConfig 복사
  4. 아래에 복사한 config를 붙여넣기
※ 동기화 키를 아는 사람 누구나 내 일기에 접근할 수 있습니다. 타인에게 공유하지 마세요.

-
※ 키를 변경하면 다른 데이터 세트로 전환됩니다. 기존 키로 돌아오면 이전 데이터를 볼 수 있습니다.
내보내기: 모든 일기를 JSON 파일로 저장합니다. 사진 포함 시 파일이 클 수 있습니다.
